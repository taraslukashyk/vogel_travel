# Brandbook: Типографіка Vogel Family Travel

Цей документ описує поточну систему шрифтів проекту та пропонує варіанти для оновлення візуального стилю.

## 1. Поточні шрифти (Current Fonts)

Зараз у проекті використовуються три основні гарнітури та системний Serif:

1.  **Inter** (Sans-serif)
    - **Використання:** Основний текст (body), цитати, опис переваг, мета-дані.
    - **Насиченість (Weights):** 300 (Light), 400 (Regular), 500 (Medium), 600 (Semi-Bold).
2.  **Montserrat** (Geometric Sans-serif)
    - **Використання:** Заголовки розділів, кнопки, акцентні плашки (badges), номери пунктів.
    - **Насиченість (Weights):** 400, 500, 600, 700 (Bold), 800 (Extra-Bold).
3.  **Marck Script** (Cursive)
    - **Використання:** Декоративні елементи заголовку в Hero-секції ("Відкрийте світ", "починається").
4.  **System Serif** (Default Serif)
    - **Використання:** Заголовки в секції "Мистецтво подорожувати" (Featured Tours) та лапки в цитатах.

## 2. Стилі та розміри (Styles & Sizes)

### Головні заголовки (Hero & Banners)
- **Акцентний скрипт (Main Hero):** 
    - Шрифт: `Marck Script`
    - Розмір: `5xl` (мобільні) → `8xl` (десктоп) / ~48px → 96px
- **Основний заголовок (Main Hero):** 
    - Шрифт: `Montserrat`, Extra-Bold, Uppercase
    - Розмір: `4xl` (мобільні) → `text-[72px]` (десктоп) / ~36px → 72px
- **Заголовок сторінки Пропозицій (Offers Hero):**
    - Шрифт: `Montserrat`, Extra-Bold, Uppercase
    - Розмір: `5xl` (мобільні) → `text-[88px]` (десктоп) / ~48px → 88px

### Заголовки розділів (Section Headers)
- **Варіант 1 (About/Mission/Offers intro):**
    - Шрифт: `Montserrat`, Bold
    - Розмір: `3xl` → `4xl` / 30px → 36px
- **Варіант 2 (Featured Tours - Decorative):**
    - Шрифт: `System Serif`, Uppercase, Tracking-widest (0.15em)
    - Розмір: `4xl` → `6xl` / 36px → 60px

### Підзаголовки та картки (Components/Cards)
- **Заголовки переваг (Advantages):**
    - Шрифт: `Montserrat`, Bold, Uppercase, Tracking-widest
    - Розмір: `xl` / 20px
- **Заголовки готелів у картках (Offer Cards):**
    - Шрифт: `Montserrat`, Bold
    - Розмір: `xl` / 20px
- **Заголовки готелів у галереї (Featured):**
    - Шрифт: `System Serif`, Italic
    - Розмір: `3xl` → `4xl` / 30px → 36px

### Основний текст (Body Text)
- **Параграфи (Intro/Description):**
    - Шрифт: `Inter`, Light/Normal
    - Розмір: `lg` (18px)
- **Текст у картках / Деталі:**
    - Шрифт: `Inter`, Light
    - Розмір: `[14px]` (0.875rem)
- **Цитати (Blockquote):**
    - Шрифт: `Inter`, Light
    - Розмір: `lg` → `xl` / 18px → 20px

### Кнопки та навігація (UI Elements)
- **Кнопки:**
    - Шрифт: `Montserrat`, Bold, Uppercase, Tracking-widest (0.15em - 0.2em)
    - Розмір: `sm` (14px) або `xs` (12px)

## 3. Професійні комбінації шрифтів (Elite Font Pairings)

Згідно з методологією **UI/UX Pro Max**, для брендів у сфері Premium Travel рекомендуються наступні поєднання, що забезпечують баланс між читабельністю та ексклюзивністю:

### Варіант А: "Timeless Elegance" (Найвища преміальність)
*Ідеально для Luxury-сегменту з акцентом на історію та якість.*
- **Заголовки (Headings):** **Playfair Display** (Serif) — високий контраст штрихів створює відчуття "глянцевого журналу".
- **Основний текст (Body):** **Inter** (Sans) — забезпечує максимальну чіткість на будь-яких пристроях.
- **Акценти:** **Tenor Sans** для дрібних підзаголовків (наприклад, локацій).
- **Mood:** Luxury, Vogue, High-end.

### Варіант Б: "Modern Boutique" (Сучасний стиль)
*Для брендів, що фокусуються на унікальному досвіді та lifestyle.*
- **Заголовки (Headings):** **Cinzel** (Serif) — шрифт з античними коріннями, що виглядає архітектурно та монументально.
- **Основний текст (Body):** **Josefin Sans** (Sans) — геометричний та легкий, що додає повітряності.
- **Акценти:** **Marck Script** (залишити поточний) для рукописних підписів.
- **Mood:** Lifestyle, Architecture, Minimalist.

### Варіант В: "Elite Modernism" (Стійка довіра)
*Фокус на надійності, професіоналізмі та сучасному сервісі.*
- **Заголовки (Headings):** **Montserrat** (Geometric Sans) — поточний вибір, але у варіації **Black/Extra-Bold** з великим трекінгом.
- **Основний текст (Body):** **Jost** (Sans) — сучасна альтернатива Futura, дуже чиста та ритмічна.
- **Акценти:** **Cormorant Garamond** (Italic) для цитат та відгуків.
- **Mood:** Reliable, Tech-forward, Professional.

## 4. UI/UX Pro Max: Найкращі практики типографіки

Щоб дизайн виглядав професійно та "дорого", слід дотримуватись наступних правил:

| Параметр | Рекомендація | Чому це важливо |
| :--- | :--- | :--- |
| **Line-height** | 1.6 - 1.8 для body | Покращує читабельність довгих описів турів. |
| **Line-length** | 65 - 75 символів | Оптимальна ширина рядка для сприйняття тексту. |
| **Letter-spacing** | +0.1em для All-Caps | Робить заголовки великими літерами більш вишуканими. |
| **Contrast** | 4.5:1 мінімум | Необхідно для доступності (accessibility) на світлих фонах. |
| **Visual Hierarchy** | Співвідношення 1.25 або 1.5 | Крок між розмірами (напр. 16px -> 24px -> 36px) створює ритм. |

## 5. Анти-патерни (Чого слід уникати)

- **Надлишок Script-шрифтів:** Використовувати `Marck Script` лише як декоративний акцент, ніколи — для довгих речень.
- **Змішування більше 3-х гарнітур:** Це перевантажує інтерфейс та робить його хаотичним.
- **Замалі відступи (Padding):** Типографіка потребує "повітря". Між текстом та краєм картки має бути мінімум `p-6` (24px).
- **Чистий чорний (#000) для тексту:** На білому фоні краще використовувати глибокий темно-сірий (наприклад, `slate-900`), це виглядає преміальніше.
